File structure



	File Structure 

	


  

  


  
In the file structure section , you can find css, fonts, img, js folder insdie the template folder and all the html files. Inside the css folder you find some vendors css file and styles.css and responsive.css files. If you want to change anything you can do it with styles.css file. If you want to optimize of change some layout on mobile device you need to change the file named responsive.css file. Suppose you want to change the color of the template, we use here the primary color '#346abb' and the hover color '#134591', sorry we can't provide less file so you need to search and replace those color with your desire color.


  

  


  
In the font folder you find some font awesome fonts file for adding fontawesome.


  

  


  
In the img folder you find all the image that used in this template, but we don't have write to distribute those image thats why we blur the image, you can change those image with you desire image, just make sure that the name of the image is same so that you don't need to change the image file name throught html file to get the image.
